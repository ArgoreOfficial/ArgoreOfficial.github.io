Common SPETS Issues

Missing .NET Core

If you're getting this error and pressing Yes does not work, download and install the .NET Core 3.1 Runtime here

This error is because SPETS requires .NET Core 3.1, which may not be installed on every computer

Missing Faces

If your imported creation is missing faces like the image above, your model is containing faces with more than 5 vertices (points). To fix this you need to triangulate your model down to at least quads.

The Importing Tutorial covers how to do it in blender.
